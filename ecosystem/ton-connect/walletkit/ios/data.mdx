Ypu are able to get wallet information, Jettons, and NFTs using the WalletKit.

<Aside type="tip">
  All methods require an existing wallet instance. [Create or retrieve](./wallets) a wallet before accessing data.
</Aside>

## Wallet information

### Balance

```swift
let balance = try await wallet.balance() // Returns balance in nanoTON
```

## Jettons

### Get all Jettons

```swift
let jettons = try await wallet.jettons(limit: TONLimitRequest(limit: 10, offset: 0))
```
### Get specific Jetton balance

```swift
let jettonAddress = "EQCxE..."
let balance = try await wallet.jettonBalance(jettonAddress) // Returns balance in nano with decimals count according to Jetton specification
println("Jetton balance: $balance")
```

### Get Jetton wallet address

```kotlin
val jettonAddress = "EQCxE..."
val walletAddress = wallet.getJettonWalletAddress(jettonAddress)
println("Jetton wallet: $walletAddress")
```

## NFTs

### Get all NFTs

```kotlin
val nfts = wallet.getNFTItems()

nfts.items.forEach { nft ->
    println("Name: ${nft.name}")
    println("Collection: ${nft.collectionName}")
    println("Address: ${nft.nftAddress}")
}
```

### Pagination

```kotlin
// First page
val page1 = wallet.getNFTItems(limit = 10, offset = 0)

// Next page
val page2 = wallet.getNFTItems(limit = 10, offset = 10)
```

### Filter by collection

```kotlin
val collectionAddress = "EQD..."
val nfts = wallet.getNFTItems(
    collectionAddress = collectionAddress
)
```

### Include indirect ownership

```kotlin
val nfts = wallet.getNFTItems(indirectOwnership = true)
```

### Get specific NFT

```kotlin
val nftAddress = "EQCA..."
val nft = wallet.getNFT(nftAddress)

nft?.let {
    println("Name: ${it.name}")
    println("Description: ${it.description}")
    println("Image: ${it.imageURL}")
    println("Owner: ${it.ownerAddress}")
    println("On sale: ${it.onSale}")
}
```
