Installation is made using Swift Package Manager.

## Adding WalletKit to XCode project

Open your project in XCode

Go to File -\> Add Package Dependencies

Search for WalletKit package by url https://github.com/ton-connect/kit-ios.git

Tap "Add Package" button

## Adding WalletKi to your Swift Package

Open your package's `Package.swift` manifest file

Add dependency:

```swift
dependencies: [
    .package(url: "https://github.com/ton-connect/kit-ios.git", .upToNextMajor(from: "0.0.2"))
]
```

Add product to your target:

```swift
.product(name: "TONWalletKit", package: "TONWalletKit")
```
