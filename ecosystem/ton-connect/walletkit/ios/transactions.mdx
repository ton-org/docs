You can create transactions for sending TONs, Jettons, and NFTs and also generate a preview of them for your app users.

<Aside type="tip">
  All methods require an existing wallet instance. [Create or retrieve](./wallets) a wallet before accessing data.
</Aside>

## Creating a transaction

Before sending any transaction, you need to create it for the appropriate contract (TON, Jetton, or NFT).

### TONs

```swift
// Transaction for 1 TON
let amountFormatter = TONTokenAmountFormatter()

// At this point you can pass an user's input
guard let amount = amountFormatter.amount(from: "1") else {
    return
}
let message = TONTransferMessage(
    toAddress: address, 
    amount: amount /* TON Wallet Address */
)
let transaction = try await wallet.transferTONTransaction(message: message)
```

### Jettons

```swift
// Transaction for 1 Jetton of some type
let amountFormatter = TONTokenAmountFormatter()
amountFormatter.nanoUnitDecimalsNumber = /* Jetton nanounits decimals */

// At this point you can pass an user's input
guard let amount = amountFormatter.amount(from: "1") else {
    return
}
let parameters = TONJettonTransferParams(
    toAddress: address, /* TON Wallet Address */
    jettonAddress: jettonAddress, /* Jetton Address */
    amount: amount
)
let transaction = try await wallet.transferJettonTransaction(parameters: parameters)
```

### NFTs

```swift
let parameters = TONNFTTransferParamsHuman(
    nftAddress: nft.address, /* NFT Address */
    toAddress: address /* TON Wallet Address */
)
let transaction = try await wallet.transferNFTTransaction(parameters: parameters)
```

## Transaction Preview

Once a transaction is created, it is best practice to present a preview of it to a user before it is actually sent.

Use created transaction to create a preview of it.

```swift
let preview = try await wallet.preview(transaction: transaction)
```

## Sending a transaction

```swift
try await wallet.send(transaction: transaction)
```
