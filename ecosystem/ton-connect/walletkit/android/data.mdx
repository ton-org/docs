Get wallet balance, jettons, and NFTs using the SDK's data retrieval API.

<Aside type="tip">
  All methods require an existing wallet instance. Create or retrieve a wallet before accessing data.
</Aside>

## Wallet information

### Balance

```kotlin
val wallet = walletKit.getWallet("EQD...")
val balance = wallet?.getBalance() // Returns balance in nanoTON
```

## Jettons

### Get all jettons

```kotlin
val jettons = wallet.getJettons()

jettons.items.forEach { jetton ->
    println("${jetton.name}: ${jetton.balance}")
    println("Symbol: ${jetton.symbol}")
    println("Decimals: ${jetton.decimals}")
}
```

### Pagination

```kotlin
// First page
val page1 = wallet.getJettons(limit = 10, offset = 0)

// Next page
val page2 = wallet.getJettons(limit = 10, offset = 10)
```

### Get specific jetton balance

```kotlin
val jettonAddress = "EQCxE..."
val balance = wallet.getJettonBalance(jettonAddress)
println("Jetton balance: $balance")
```

### Get jetton wallet address

```kotlin
val jettonAddress = "EQCxE..."
val walletAddress = wallet.getJettonWalletAddress(jettonAddress)
println("Jetton wallet: $walletAddress")
```

## NFTs

### Get all NFTs

```kotlin
val nfts = wallet.getNFTItems()

nfts.items.forEach { nft ->
    println("Name: ${nft.name}")
    println("Collection: ${nft.collectionName}")
    println("Address: ${nft.nftAddress}")
}
```

### Pagination

```kotlin
// First page
val page1 = wallet.getNFTItems(limit = 10, offset = 0)

// Next page
val page2 = wallet.getNFTItems(limit = 10, offset = 10)
```

### Filter by collection

```kotlin
val collectionAddress = "EQD..."
val nfts = wallet.getNFTItems(
    collectionAddress = collectionAddress
)
```

### Include indirect ownership

```kotlin
val nfts = wallet.getNFTItems(indirectOwnership = true)
```

### Get specific NFT

```kotlin
val nftAddress = "EQCA..."
val nft = wallet.getNFT(nftAddress)

nft?.let {
    println("Name: ${it.name}")
    println("Description: ${it.description}")
    println("Image: ${it.imageURL}")
    println("Owner: ${it.ownerAddress}")
    println("On sale: ${it.onSale}")
}
```
