Get wallet balance, jettons, and NFTs using the SDK's data retrieval API.

<Aside type="tip">
  All methods require an existing wallet instance. Create or retrieve a wallet before accessing data.
</Aside>

## Wallet information

### Balance

```kotlin
val wallet = walletKit.getWallet("EQD...")
val balance = wallet?.getBalance() // Returns balance in nanoTON
```

## Jettons

### Get all jettons

```kotlin
val jettons = wallet.getJettons()

jettons.items.forEach { jetton ->
    println("${jetton.name}: ${jetton.balance}")
    println("Symbol: ${jetton.symbol}")
    println("Decimals: ${jetton.decimals}")
}
```

### Pagination

```kotlin
// First page
val page1 = wallet.getJettons(limit = 10, offset = 0)

// Next page
val page2 = wallet.getJettons(limit = 10, offset = 10)
```

### Get specific jetton balance

```kotlin
val jettonAddress = "EQCxE..."
val balance = wallet.getJettonBalance(jettonAddress)
println("Jetton balance: $balance")
```

### Get jetton wallet address

```kotlin
val jettonAddress = "EQCxE..."
val walletAddress = wallet.getJettonWalletAddress(jettonAddress)
println("Jetton wallet: $walletAddress")
```

## NFTs

### Get all NFTs

```kotlin
val nfts = wallet.getNFTItems()

nfts.items.forEach { nft ->
    println("Name: ${nft.name}")
    println("Collection: ${nft.collectionName}")
    println("Address: ${nft.nftAddress}")
}
```

### Pagination

```kotlin
// First page
val page1 = wallet.getNFTItems(limit = 10, offset = 0)

// Next page
val page2 = wallet.getNFTItems(limit = 10, offset = 10)
```

### Filter by collection

```kotlin
val collectionAddress = "EQD..."
val nfts = wallet.getNFTItems(
    collectionAddress = collectionAddress
)
```

### Include indirect ownership

```kotlin
val nfts = wallet.getNFTItems(indirectOwnership = true)
```

### Get specific NFT

```kotlin
val nftAddress = "EQCA..."
val nft = wallet.getNFT(nftAddress)

nft?.let {
    println("Name: ${it.name}")
    println("Description: ${it.description}")
    println("Image: ${it.imageURL}")
    println("Owner: ${it.ownerAddress}")
    println("On sale: ${it.onSale}")
}
```

## Jettons

### Getting all jettons

Retrieve all jettons held by a wallet:

```kotlin
val jettons = wallet.getJettons()

jettons.items.forEach { jettonWallet ->
    println("Jetton: ${jettonWallet.name}")
    println("Symbol: ${jettonWallet.symbol}")
    println("Balance: ${jettonWallet.balance}")
    println("Decimals: ${jettonWallet.decimals}")
}
```

With pagination:

```kotlin
val jettons = wallet.getJettons(
    limit = 50,
    offset = 0
)
```

### Jetton metadata

Access comprehensive metadata:

```kotlin
jettonWallet.let {
    val name = it.name                    // "Tether USD"
    val symbol = it.symbol                // "USD₮"
    val decimals = it.decimals            // 6
    val image = it.image                  // Logo URL
    val description = it.description      // Token description
    val verification = it.verification    // Verification status
}
```

### Specific jetton balance

Query balance for a specific jetton:

```kotlin
val jettonAddress = "EQCxE6mUtQJKFnGfaROTKOt1lZbDiiX1kCixRv7Nw2Id_sDs"
val balance = wallet.getJettonBalance(jettonAddress)
```

### Jetton wallet address

Get the jetton wallet contract address:

```kotlin
val jettonMasterAddress = "EQCxE6mUtQJKFnGfaROTKOt1lZbDiiX1kCixRv7Nw2Id_sDs"
val jettonWalletAddress = wallet.getJettonWalletAddress(jettonMasterAddress)
```

### Formatting jetton amounts

Convert raw balance using decimals:

```kotlin
fun formatJettonBalance(balance: String, decimals: Int): String {
    val balanceValue = balance.toBigDecimal()
    val divisor = BigDecimal.TEN.pow(decimals)
    val formatted = balanceValue.divide(divisor)
    return formatted.toPlainString()
}

val rawBalance = jettonWallet.balance ?: "0"
val decimals = jettonWallet.decimals ?: 9
val formatted = formatJettonBalance(rawBalance, decimals)
```

## NFTs

### Getting all NFTs

Retrieve NFT items:

```kotlin
val nfts = wallet.getNFTItems()

nfts.items.forEach { nft ->
    println("NFT: ${nft.metadata?.name}")
    println("Address: ${nft.address}")
    println("Image: ${nft.metadata?.image}")
}
```

With pagination:

```kotlin
val nfts = wallet.getNFTItems(
    limit = 50,
    offset = 0
)
```

### NFT metadata

Access NFT metadata:

```kotlin
nft.metadata?.let { metadata ->
    val name = metadata.name              // NFT name
    val description = metadata.description // NFT description
    val image = metadata.image            // Image URL
    val attributes = metadata.attributes   // NFT attributes/traits
}
```

### Collection information

Get collection details:

```kotlin
nft.collection?.let { collection ->
    val address = collection.address           // Collection contract address
    val nextItemIndex = collection.nextItemIndex // Next item to be minted
    val owner = collection.ownerAddress        // Collection owner
}
```

Filter by collection:

```kotlin
val collectionAddress = "EQDWq..."
val nfts = wallet.getNFTItems(
    collectionAddress = collectionAddress
)
```

### Getting specific NFT

Retrieve a single NFT:

```kotlin
val nftAddress = "EQCA..."
val nft = wallet.getNFT(nftAddress)

nft?.let {
    println("Owner: ${it.ownerAddress}")
    println("On Sale: ${it.onSale}")
}
```

### Sale and auction information

Check NFT status:

```kotlin
nft.let {
    if (it.onSale == true) {
        println("On sale: ${it.saleContractAddress}")
    }
    
    it.auctionContractAddress?.let { auction ->
        println("On auction: $auction")
    }
}
```

## Error handling

Handle network errors:

```kotlin
try {
    val balance = wallet.getBalance()
    val jettons = wallet.getJettons()
    val nfts = wallet.getNFTItems()
} catch (e: Exception) {
    println("Failed to fetch data: ${e.message}")
}
```

## Best practices

1. **Cache data** — Balance, jettons, and NFTs don't change frequently
2. **Use pagination** — Load large collections in batches
3. **Handle precision** — Use `BigDecimal` for financial calculations
4. **Check decimals** — Always verify decimals before displaying amounts
5. **Handle missing metadata** — Some tokens may have incomplete data
