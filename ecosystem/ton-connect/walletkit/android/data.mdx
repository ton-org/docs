Retrieve wallet information, jettons, and NFTs using WalletKit.

<Aside type="tip">
  All methods require an existing wallet instance. [Create or retrieve](./wallets) a wallet before accessing data.
</Aside>

## Wallet information

### Balance

```kotlin
// Returns balance in nanoTON
val balance = wallet.getBalance()
```

### Address

```kotlin
val address = wallet.address
```

## Jettons

### Get all Jettons

```kotlin
val jettons = wallet.getJettons(limit = 10, offset = 0)
```

### Get specific Jetton balance

```kotlin
val jettonAddress = "EQC_REPLACE_WITH_JETTON_ADDRESS"
// Returns balance in nano with decimals count according to Jetton specification
val balance = wallet.getJettonBalance(jettonAddress) 
```

### Get Jetton wallet address

```kotlin
val jettonAddress = "EQCxE..."
val walletAddress = wallet.getJettonWalletAddress(jettonAddress)
```

## NFTs

### Get all NFTs

```kotlin
val nfts = wallet.getNFTItems(limit = 10, offset = 0)
```

### Get specific NFT

```kotlin
val nftAddress = "EQCA..."
val nft = wallet.getNFT(nftAddress)
```
