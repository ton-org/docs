Send jetton and NFT transfers using the SDK's transaction creation and sending API.

<Aside type="tip">
  All transfers require two steps: creating the transaction and sending it to the blockchain.
</Aside>

## Jetton transfers

### Basic transfer

```kotlin
val wallet = walletKit.getWallet("EQD...")

val params = TONJettonTransferParams(
    toAddress = "EQD...",
    jettonAddress = "EQCxE...",
    amount = "1000000",
    comment = "Payment"
)

val transactionContent = wallet?.createTransferJettonTransaction(params)
val txHash = transactionContent?.let { wallet.sendTransaction(it) }
```

## NFT transfers

### Basic transfer

```kotlin
val wallet = walletKit.getWallet("EQD...")

val params = TONNFTTransferParamsHuman(
    nftAddress = "EQCA...",
    transferAmount = "50000000",    // 0.05 TON for gas
    toAddress = "EQD...",
    comment = "Gift"
)

val transactionContent = wallet?.createTransferNFTTransaction(params)
val txHash = transactionContent?.let { wallet.sendTransaction(it) }
```

<Aside type="note">
  NFT transfers require TON for gas fees (typically 0.05 TON).
</Aside>

## Best practices

1. **Validate addresses** — Verify recipient address format
1. **Check balance** — Ensure sufficient balance before transfer
1. **Use correct decimals** — Different jettons have different decimal places
1. **Verify ownership** — Check NFT ownership before transfer
1. **Handle errors** — Network issues can cause failures
1. **Keep transaction hash** — Store for tracking
