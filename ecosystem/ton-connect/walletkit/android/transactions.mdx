You are able to create transaction for sending TONs, Jettons and NFTs and also generate preview of it for your app users.

<Aside type="tip">
  All methods require an existing wallet instance. [Create or retrieve](./wallets) a wallet before accessing data.
</Aside>

## Creating a transaction

Before sending any transaction you need to create it for appropriate contract (TON, Jetton or NFT).

### TONs

```kotlin
val message = TONTransferParams(
    toAddress = "EQD...", /* TON Wallet Address */
    amount = "1000000000" /* 1 TON in nanoTON */
)
val transaction = wallet.createTransferTonTransaction(message)
```

### Jettons

```kotlin
val parameters = TONJettonTransferParams(
    toAddress = "EQD...", /* TON Wallet Address */
    jettonAddress = "EQCxE...", /* Jetton Address */
    amount = "1000000" /* Amount in jetton units */
)
val transaction = wallet.createTransferJettonTransaction(parameters)
```

### NFTs

```kotlin
val parameters = TONNFTTransferParamsHuman(
    nftAddress = "EQCA...", /* NFT Address */
    toAddress = "EQD..." /* TON Wallet Address */
)
val transaction = wallet.createTransferNFTTransaction(parameters)
```

## Transaction Preview

Once transaction is created it is a best practice to present a preview of it to a user before it actually sent.

Use created transaction to create a preview of it.

```kotlin
val preview = wallet.getTransactionPreview(transaction)
```

## Sending a transaction

```kotlin
wallet.sendTransaction(transaction)
```
