Installation is made using Gradle dependency management.

## Requirements

- Android SDK 24 (Android 7.0) or higher
- Java 17 or higher
- Gradle 7.0 or higher

## Adding the dependency

Add the dependency to your app's `build.gradle.kts` file:

```kotlin
dependencies {
    implementation("io.ton:walletkit:0.1.0")
}
```

Ensure Maven Central is included in your repository list:

```kotlin
repositories {
    google()
    mavenCentral()
}
```

## Permissions

Add the required permission to your `AndroidManifest.xml`:

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android">
    
    <uses-permission android:name="android.permission.INTERNET" />
    
    <!-- Your application configuration -->
    
</manifest>
```

This permission is required for the SDK to communicate with dApps and the TON blockchain.
