## Overview

All TON Center API v2 methods use a standard set of HTTP status codes to indicate the result of a request.

| Status | Meaning                                             | Description                                                                                 |
| ------ | --------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| `404`  | Data not found                                      | The requested resource does not exist in storage.                                           |
| `405`  | Wrong HTTP method                                   | The endpoint was called with an unsupported HTTP method. Use GET or POST.                   |
| `409`  | Resource exists but wrong type                      | The resource was found but does not match the expected type for this method.                |
| `422`  | Invalid request parameters                          | The request parameters failed validation (e.g., missing, malformed, or conflicting values). |
| `429`  | Rate limit exceeded                                 | Too many requests. Back off and retry.                                                      |
| `500`  | Server-side failure                                 | An internal error occurred. Retry or contact support if it persists.                        |
| `504`  | Timeout waiting for liteserver                      | The liteserver did not respond in time. Retry the request.                                  |
| `542`  | Liteserver-specific error or unsupported stack type | A liteserver error or an unsupported TVM stack type was encountered.                        |

For method-specific error messages and troubleshooting details, refer to the documentation for the relevant endpoint.
