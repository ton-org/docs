Metadata is split into two parts: the collection stores collection‑wide data, and the item stores item‑specific data (not necessarily the full metadata).

### High level

There is a TON Center [API method](/ecosystem/rpc/toncenter/v3/accounts/get-account-metadata?playground=open) that retrieves metadata.

### Low level

To get full NFT metadata:

1. Resolve the NFT item address by index from the collection (if needed) using [get\_nft\_address\_by\_index(index)](/standard/tokens/nft/api#get-nft-address-by-index).
1. Get the item's individual content from the item contract using [get\_nft\_data()](/standard/tokens/nft/api#get-nft-data).
1. Get the full metadata from the collection contract using [get\_nft\_content(index, individual\_content)](/standard/tokens/nft/api#get-nft-content).

[See more about metadata.](/standard/tokens/metadata)
