Tolk is a statically typed language for writing smart contracts on TON. It provides declarative data structures, automatic cell serialization, and first-class message handling.

The language compiles to [TVM](/tvm/overview) and provides direct control over execution.

```tolk
type AllowedMessage = CounterIncrement | CounterReset

fun onInternalMessage(in: InMessage) {
    val msg = lazy AllowedMessage.fromSlice(in.body);
    match (msg) {
        CounterIncrement => { ... }
        CounterReset => { ... }
    }
}

get fun currentCounter() {
    val storage = lazy Storage.load();
    return storage.counter;
}
```

Tolk is compatible with existing [TON standards](/from-ethereum#standards).

## Key features

Tolk provides high-level readability while preserving low-level control:

- a type system for describing [cell](/foundations/serialization/cells) layouts;
- [`lazy` loading](/languages/tolk/features/lazy-loading) that skips unused fields;
- unified message composition and deployment;
- a compiler targeting the Fift assembler;
- tooling with IDE integration.

## From FunC to Tolk

Tolk evolved from FunC and is now the recommended language for TON smart contracts. To migrate from FunC:

- check [gas benchmarks](https://github.com/ton-blockchain/tolk-bench);
- read [Tolk vs FunC](/languages/tolk/from-func/tolk-vs-func) for an overview;
- use the [FunC-to-Tolk converter](/languages/tolk/from-func/converter) to migrate existing projects.

## Quick start

1. Run the command:

   ```bash
   npm create ton@latest
   ```

1. Enter a project name and choose "simple counter contract".

1. Follow [Your first smart contract](/contract-dev/first-smart-contract) page to write the contract in Tolk.

## IDE support

1. [JetBrains IDEs plugin](/contract-dev/ide/jetbrains#tolk) provides syntax highlighting, code navigation, and more. Download from [marketplace](https://plugins.jetbrains.com/plugin/23382-ton) and [install it](/contract-dev/ide/jetbrains#installation).
1. [VSCode extension](/contract-dev/ide/vscode#tolk) provides feature-rich extension for VSCode and VSCode-based editors like VSCodium, Cursor, Windsurf, and others.
   - Download from [marketplace](https://marketplace.visualstudio.com/items?itemName=ton-core.vscode-ton) and [install it](/contract-dev/ide/vscode#from-visual-studio-marketplace).
   - Download from [OpenVSX registry](https://open-vsx.org/extension/ton-core/vscode-ton) and [install it](/contract-dev/ide/vscode#from-open-vsx-registry).
1. [Language server](https://github.com/ton-blockchain/ton-language-server#other-editors) supports (Neo)Vim, Helix, and other editors with LSP support. Download an archive and [install it](/contract-dev/ide/vscode#from-disk).
